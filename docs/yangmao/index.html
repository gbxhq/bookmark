下面是一群建立时的公告

本着互联网的分享精神，我打算建一个薅羊毛的社群。

具体有什么内容呢？看一下成果：

建群半个月撸过的经典羊毛

  • 1月7日，Airbnb重大Bug：换币种支付不变价格。英镑下单日元支付0.1折住酒店。
  • 1月9日，京东钟表神券101-100. 几个店铺因此倒闭关店。
  • 1月20日，拼多多神券100-100。群里大部分人0撸100话费。有人撸上千QB。此事已上头条不多介绍。

后来撸过的太多了。至2019年5月一群人满。全靠群里的人口口相传。不多宣传了。

入群方式：

扫码关注，点击菜单中的【自助拉群】会弹出机器人微信号。添加即可。

机器人有时候不稳定不能拉群。试着对机器人发送"羊毛"。

群规

  • 扫码入群，目前人员较少。若超过一定人数开始采用邀请制入群。

  • 防止消息被覆盖，群内不要闲聊。尽量只讨论相关话题。

  • 禁止发布带有个人邀请的线报。发一条送飞机票。

  • 群内发言触发关键词（敏感、广告、返利、推广等）、url、名片、二维码、小程序自动踢人，且进入黑名单，黑名单的人无法进入我的任何一个群。

  • 为了给大家一个干净的线报环境。无论是不是群主分享的活动，都不要把链接重新发到本群。发一次送飞机票。

  • 拒绝利润，只是有些活动需要助力，你们能给我凑凑人头就够了。

感谢！

群内线报类型介绍

薅羊毛只是娱乐，薅一下开心，没赶上也无所谓。

可以拉些朋友进群一起哦，这样遇到好东西多一双眼睛盯着！

  • Bug单，顾名思义，由于商家、平台设置错误，导致的重大羊毛，这种bug单可遇而不可求。

  • 双叠加，这种双叠加类型也很经典了。举例说明吧，商品A可以用199-100的优惠券，同时A还参加“满两件打5折的活动”，这时候我们拍N个A，凑够199，就可以做到，199-100-100. 这就叫双叠加。

    问：哪里找双叠加？

    答：盯好群，双叠加出现的时间一般是每天的0点。隔一段时间会出。

  • 消灭运费券，主要是些1元的自营商品，plus会员每月送的6张运费券用不掉可以上车。出现时间一般是每天10点、16点、20点等。

  • 商家套全品，plus会员每个月还有100块的全品，用不了怎么办？我来帮你，这些全品券有第三方商家会去套，他们会把商品价格设的很高，让你刚好领了券可以花1块、2块的低价买掉他的商品，这样我们低价买东西，他们套到全品券跟jd平台兑现，互惠互利。

除了jd线报，群内还会分享各种线报活动。不多介绍了！

线报都是爬虫爬回来的，京东自营放心买，有的是京东第三方的，大家要有鉴别能力，不要盲目购买！要有鉴别能力，不要盲目购买！要有鉴别能力，不要盲目购买！

统一回复

淘宝的白菜价网站已经泛滥了，我不怎么发淘宝商品就是因为没法为大家做好品控。

我发的多是京东自营/旗舰店商品，产生了退换纠纷，售后服务真的比淘宝好太多。

别让你们的父母、朋友在一些“x购”“xx生活”的淘宝白菜群了，来买京东，不翻车。
