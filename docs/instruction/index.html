推荐你这么用炫猿

炫猿导航收集网址的主要原则就是看网站的实用性。

目前网络上的优秀导航站有很多，比如龙轩导航。收集的很全。

炫猿导航还是比较小众。不能满足大部分朋友的需要。

因此我建议你这么使用炫猿导航：

浏览器插件推荐

使用Infinity新标签页的浏览器插件。支持Chrome内核和Firefox。

这里给一个下载地址。

https://extfans.com/extension/dbfmnekepjoapopniengjbcpnbljalfg

具体用法

有了这个插件有啥用呢？首先它内置了超多【正牌】网站，就是一些知名度比较高的网站（这些网站我一般不收录）。

你就可以把个人经常访问的一些网站、网址自定义地添加到新的标签页里。

而且这款插件支持同步，以后换电脑啥的都不用担心书签的丢失。
